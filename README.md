# BOSS直聘消息与简历处理技能

## 简介

这是一个用于 BOSS 直聘企业端的自动化招聘技能，聚焦消息收口、简历评估、先判定再下载、飞书招聘批量导入闭环，并支持将结构化结果同步到飞书多维表格（Bitable）作为数据层。

## 核心能力

- 消息处理：先清 `沟通中/全部`，再分视角推进状态
- 严格评估：按岗位缓存规则执行，硬性条件先判
- 先判定再下载：通过前不下载，避免无效文件
- 飞书招聘闭环：批量新建 -> 缺必填项修复 -> 重复人才处理 -> 提交
- Bitable 同步（可选）：候选人主档、交互流水、职位漏斗看板

## 关键规则（节选）

- 默认顺序：`先处理消息 -> 再处理今天简历 -> 条件满足则上传飞书`
- BOSS 不通过：统一点击 `其他原因`
- 上传前二次本科硬闸：`undergraduate_passed=true` 才可上传
- 自动上传触发条件：`本轮新增通过 > 0 && 已下载待上传 > 0`

完整规则见 [SKILL.md](./SKILL.md)。
