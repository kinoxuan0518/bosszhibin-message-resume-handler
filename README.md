# BOSS直聘消息与简历处理技能

## 简介

这是一个用于 BOSS 直聘的自动化招聘技能，专门处理候选人消息、简历评估、下载和飞书招聘系统批量导入。

## 核心功能

- 📨 **消息处理**：自动扫描和处理 BOSS 直聘会话
- 📄 **简历请求**：智能请求和接收候选人简历
- ⚖️ **先判定再下载**：基于岗位规则评估候选人，避免无效下载
- 📥 **批量导入飞书**：自动上传合格简历到飞书招聘系统
- 🔧 **异常处理**：自动处理"缺必填项"和"重复人才"状态

## 工作流程

```
消息扫描 → 请求/接收简历 → 预判评估 → 下载简历（仅通过者）
→ 批量上传飞书 → 处理异常状态 → 提交完成
```

## 评估标准（100分制）

### 硬性条件（必须通过）
- 学历门槛
- 工作年限门槛
- 核心技能门槛

### 权重评分
- 教育背景：25分
- 工作经历与公司背景：25分
- 核心技能匹配：30分
- 领域经验与项目产出：20分

## 与其他技能协作

本技能与 `bosszhibin-auto-recruiter` 技能协作：
- **auto-recruiter**：负责职位解析、规则缓存、主动打招呼
- **本技能**：负责消息收口、严格判定、下载、飞书导入

## 特性

- ✅ 状态机管理候选人处理流程
- ✅ 先判定再下载，减少无效操作
- ✅ 幂等性设计，避免重复处理
- ✅ 自动处理飞书导入异常状态
- ✅ 完整的缓存机制

## 缓存文件

- 位置：`./bosszhibin_resume_cache.json`
- 记录：候选人处理状态、评分、决策来源

## 触发命令

- `处理今天简历`
- `处理消息`
- `处理简历`
- `批量上传飞书`
- `批量处理简历`
